# GRMP Demonstrators

A repository containing demonstration config files of the GRMP test orchestration system which runs test workers in isolated containers and combines their JUnit XML reports.

The orchestrator can be found [HERE](https://github.com/vliz-be-opsci/grmp-framework) while possible tests can be found [HERE](https://github.com/vliz-be-opsci/grmp-test-implementations).
